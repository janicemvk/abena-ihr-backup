# Abena eCBome Analyzer

A comprehensive health analysis and optimization system that focuses on the endocannabinoidome (eCBome) system. This React-based application provides detailed insights into various health metrics, generates reports, and offers personalized recommendations for health optimization.

## Features

- Real-time health metrics monitoring
- Interactive data visualization with charts
- Comprehensive health analysis
- Personalized recommendations
- HTML report generation
- Drug interaction analysis
- Genetic polymorphism tracking
- Microbiome integration
- Treatment modality suggestions

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/abena-ecbome-analyzer.git
cd abena-ecbome-analyzer
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Usage

1. **Health Metrics Display**
   - View real-time health metrics including stress level, sleep quality, and physical activity
   - Monitor overall health score
   - Track historical data through interactive charts

2. **Analysis and Recommendations**
   - Get personalized health recommendations based on current metrics
   - View detailed analysis of your eCBome system
   - Receive prioritized action items for health optimization

3. **Report Generation**
   - Generate comprehensive HTML reports
   - Include detailed analysis and recommendations
   - Export and share reports with healthcare providers
