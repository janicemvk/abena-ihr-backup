# ✅ Admin Portal - Using Port 3010

## 🎯 **Port Changed to 3010**

Ports 3000-3005 are all in use, so I've updated the admin portal to use **port 3010** (higher port, less likely to conflict).

## 🚀 **Start the Admin Portal**

```powershell
cd "c:\Next-JS\admin 2\Project Admin"
npm run dev
```

## 🌐 **Access the Admin Portal**

- **Home Page:** http://localhost:3010
- **Admin Dashboard:** http://localhost:3010/admin
- **Users:** http://localhost:3010/admin/users
- **Appointments:** http://localhost:3010/admin/appointments
