# WebSocket API

Real-time WebSocket API for the ABENA blockchain: block and event subscriptions, notifications.

**Planned:**
- New block / finalization subscriptions
- Pallet event streams (e.g. `HealthRecordHash`, `PatientIdentity`, `AbenaCoin`)
- Notifications for record access, consent changes, quantum result attestations

Integrates with `node` RPC (WebSocket transport).
